  • unwrap(): RawProbeStream

  • Inherited from BaseStream.unwrap

    Unwrap the enhanced instance back to its untouched plain object form. This is useful if you need to access properties which are not (yet) wrapped. Please open an issue or a pull request to wrap those properties ;)

const rawStream = stream.unwrap();
